ECMAScript 2015, or ES6, is a standardized version of JavaScript released in 2015. It added a huge update to the language since ES5, containing new features and different syntax that enables developers to write shorter code for enhanced functionnalities. While you can still write programs in ES5, ES6 added many powerful new features to the language:

  • Variable declaration
  • Arrow functions
  • Array prototypes
  • String prototypes
  • template literals
  • Spreading
  • Destructuring
  • Import
  • Export
